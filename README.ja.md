# ft_containers

STLコンテナの再実装です。

## 概要

C++98で以下のコンテナを再実装しました。

- vector
- map
- stack
- set

## テスト

全てのコンテナのテスト

```bash
make test
```

各コンテナのテスト

```bash
./grademe.sh vector
```
