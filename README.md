# ft_containers

Reimplementation of STL containers.

## Overview

I reimplemented the following containers in C++98.

- vector
- map
- stack
- set

## Testing

Test all containers

```bash
make test
```

Test the container specified by the argument

```bash
./grademe.sh vector
```
